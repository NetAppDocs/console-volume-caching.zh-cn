= 在审计页面上监控卷缓存作业
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用NetApp Console审核页面监控所有卷缓存作业并确定其进度。

.步骤
. 从NetApp Console菜单中，选择 *管理* > *审计*。
. （可选）按时间、服务、操作、代理、资源、用户或状态进行过滤。
. 探索所有缓存和操作。
